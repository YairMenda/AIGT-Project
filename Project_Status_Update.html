Project Status Update

Project Title: Comparative Analysis of Counterfactual Regret Minimization (CFR) Variants in Kuhn Poker

Students: Yair Menda (207273558), Roy Houber (322274259)

Date: December 14, 2025

(a) What has been done so far

We have successfully completed the implementation phase of our proposed research and have begun the analysis phase. Key milestones achieved include:

  • Done Game Engine Implementation: Developed a fully functional custom simulator for Kuhn Poker, including game tree generation, information set handling, and terminal state evaluation.
  • Done Algorithm Implementation: Implemented all proposed solver variants:
    • Vanilla CFR (Baseline)
    • CFR+ (with regret flooring and alternating updates)
    • Discounted CFR (DCFR) with configurable parameters
    • Linear CFR
  • Done Extended Scope: Implemented additional experimental solvers not originally planned, including Quadratic CFR, Exponential CFR, and Pruned CFR, to broaden our comparative analysis.
  • Done Analysis Framework: Built a robust benchmarking suite (analysis.py) that tracks:
    • Exploitability (Nash distance) over iterations.
    • Memory usage estimates for each solver.
    • Strategy profiles at key decision points.
  • Done Preliminary Results: Generated initial comparative graphs (Exploitability vs. Iterations) confirming that CFR+ and DCFR significantly outperform the Vanilla CFR baseline in convergence speed.

(b) What is left to do

Our focus now shifts to deep analysis and final reporting. The remaining tasks are:

  • To Do High-Fidelity Experiments: Run long-duration simulations (high iteration counts) to observe tail-end convergence behavior and stability for all variants.
  • To Do Strategy Profile Analysis: Specifically analyze the final mixed strategies (e.g., bluffing frequencies) to verify if all solvers converge to the exact same GTO frequencies or if they find different valid equilibria.
  • To Do Parameter Sensitivity: Briefly test sensitivity of DCFR to its discount parameters ($\alpha, \beta, \gamma$).
  • To Do Final Report: Synthesize all findings into the final project report, interpreting the theoretical implications of the empirical results.
